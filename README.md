 Aegis Guard - Phishing URL Detector

A machine learning-based phishing URL detection system developed using Python, Streamlit, and Random Forest.

 Features
- Phishing URL Detection
- Threat Score Analysis
- Interactive Dashboard
- Scan History Tracking

 Technologies Used
- Python
- Streamlit
- Scikit-Learn
- Pandas
- Joblib
- XGBoost
  

 How to Run

pip install -r requirements.txt

streamlit run URL_phishing.py
